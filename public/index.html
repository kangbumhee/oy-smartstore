AI API 키 발급 방법 보기

방법 1: Google Gemini (무료)

  1. Google AI Studio에 접속합니다.
  2. Google 계정으로 로그인 후 Get API Key 클릭합니다.
  3. 생성된 API Key (AIzaSy...)를 복사합니다.
  4. 아래 AI API Key에 붙여넣고, Base URL은 비워둡니다.

방법 2: OpenAI 호환 API (claudeagent 등)

  1. claudeagent.com.cn에 회원가입합니다.
  2. 콘솔 → 令牌管理(토큰 관리) → 새 토큰 생성합니다.
  3. 생성된 키 (sk-xxx)를 AI API Key에 입력합니다.
  4. Base URL에 https://claudeagent.com.cn/v1을 입력합니다.
  5. 모델에서 claude-sonnet-4-6을 선택합니다.
